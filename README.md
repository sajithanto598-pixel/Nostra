# 🛍️ NOSTRA – Fashion E-Commerce Website

Nostra is a modern and responsive fashion e-commerce website designed to provide a seamless shopping experience. It showcases trendy collections, interactive UI elements, and clean design principles suitable for real-world web applications.

---

## 🚀 Features

- 🖼️ Interactive image slider (hero section)
- 🛒 Product showcase (New Arrivals & Most Wanted)
- 🔍 Search and filter functionality
- 📱 Fully responsive design (mobile-friendly)
- 📩 Contact page with form validation
- 🎨 Clean and modern UI/UX
- 📦 Organized structure for scalability

---

## 🧰 Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & Layout
- **JavaScript (Vanilla)** – Interactivity
